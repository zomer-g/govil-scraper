                     תיעוד API
                

                

                    

                        כל נקודות הקצה של האוספים והקבצים הן ציבוריות ולא דורשות הזדהות.
                        סריקה חדשה דורשת התחברות כמנהל (Google SSO).
                    


                    
אוספים (ציבורי)


                    

                        

                            GET
                            /api/collections
                        

                        רשימת כל האוספים — כולל מקור, תאריך סריקה, מספר רשומות וקבצים
                    


                    

                        

                            GET
                            /api/collections/{id}
                        

                        פרטי אוסף + תצוגה מקדימה של 50 שורות ראשונות
                    


                    

                        

                            GET
                            /api/collections/{id}/download
                        

                        הורדת קובץ ZIP (נוצר דינמית מקבצים בודדים)
                    


                    

                        

                            GET
                            /api/collections/{id}/csv
                        

                        הורדת קובץ CSV בלבד
                    


                    

                        

                            GET
                            /api/collections/{id}/excel
                        

                        הורדת קובץ Excel בלבד
                    


                    
קבצים בודדים (ציבורי)


                    

                        

                            GET
                            /api/collections/{id}/files
                        

                        רשימת כל הקבצים באוסף — כולל שם, סוג, גודל וקישור הורדה
                    


                    

                        

                            GET
                            /api/collections/{id}/files/{filename}
                        

                        הורדת קובץ בודד מאוסף
                    


                    

                        

                            GET
                            /api/files?q={search}
                        

                        חיפוש קבצים לפי שם בכל האוספים (מינימום 2 תווים)
                    


                    
סריקה והעלאה (מנהל בלבד — דורש Google SSO)


                    

                        

                            POST
                            /api/scrape
                        

                        התחלת סריקה חדשה. מוגבל ל-5 בקשות בדקה.
                        
Body: { "url": "https://www.gov.il/he/...", "download_files": true }

                    


                    

                        

                            POST
                            /api/collections/upload
                        

                        העלאת סריקה מקומית (ZIP). קבצים נשמרים בנפרד. מוגבל ל-10 בקשות בדקה, עד 200MB.
                        
Content-Type: multipart/form-data
Fields:
  file:           ZIP file (required)
  source_url:     original gov.il URL (required)
  collector_name: display name (optional)

                    


                    

                        

                            GET
                            /api/progress/{job_id}
                        

                        SSE stream של התקדמות הסריקה
                    


                    

                        

                            GET
                            /api/status/{job_id}
                        

                        סטטוס סריקה (polling fallback)
                    


                    

                        

                            DELETE
                            /api/collections/{id}
                        

                        מחיקת אוסף (מנהל בלבד)
